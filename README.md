# AuctionX API

## Requirements

- PHP >= 8.0.2

## Install locally

- Install Docker https://docs.docker.com/get-docker/
- `cp .env .env.local`
- `docker-compose build --no-cache --pull`

## Run locally

- `docker-compose up -d`
- Visit https://api.auctionx.localhost/
